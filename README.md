## Credit Default Risk Predictor

End-to-end ML application for predicting credit default risk using an XGBoost model with a Streamlit interface.

### Features
- Trained XGBoost classifier for credit default prediction
- Consistent preprocessing pipeline
- Custom probability thresholding
- Streamlit-based interactive UI

### Tech Stack
- Python
- XGBoost
- scikit-learn
- Streamlit

### How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
